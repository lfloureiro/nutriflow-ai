# NutriFlow AI

Protótipo inicial do sistema de planeamento alimentar, gestão de despensa e geração automática de listas de compras.

## Estrutura
- `backend/` API e lógica de negócio
- `frontend/` interface web
- `infra/` docker, compose e configuração
- `docs/` documentação
- `scripts/` scripts auxiliares
